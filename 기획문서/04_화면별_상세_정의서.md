# 메뉴얼 제작 서비스 - 화면별 상세 정의서

---

## 1. 회원가입/로그인 영역

### 1-1. 회원가입 페이지

**목적**
- 새로운 사용자의 계정을 생성하고 서비스 이용을 시작

**사용자 행동**
- 이메일, 비밀번호, 이름 입력
- 이용약관 및 개인정보처리방침 동의
- 회원가입 버튼 클릭

**주요 구성요소**
- 이메일 입력 필드 (중복 확인)
- 비밀번호 입력 필드 (최소 8자, 영문+숫자)
- 비밀번호 확인 필드 (일치 여부 실시간 검증)
- 이름 입력 필드
- 약관 동의 체크박스 (필수)
- 가입하기 버튼
- 로그인 페이지로 이동 링크

**다음 단계**
- 초대 링크로 가입한 경우 → 해당 팀 자동 가입 후 대시보드
- 일반 가입인 경우 → 팀 생성 또는 초대 대기 안내 페이지

**예외처리**
- 이메일 중복: "이미 사용 중인 이메일입니다" 에러 표시
- 비밀번호 불일치: "비밀번호가 일치하지 않습니다" 에러 표시
- 약관 미동의: 가입 버튼 비활성화

---

### 1-2. 로그인 페이지

**목적**
- 기존 사용자의 계정 인증 및 서비스 접속

**사용자 행동**
- 이메일, 비밀번호 입력
- 로그인 버튼 클릭

**주요 구성요소**
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 비밀번호 찾기 링크
- 회원가입 링크

**다음 단계**
- 팀 소속 사용자 → 대시보드
- 팀 미소속 사용자 → 팀 생성/초대 안내 페이지

**예외처리**
- 로그인 실패: "이메일 또는 비밀번호가 일치하지 않습니다" 에러 표시
- 5회 실패 시: 계정 잠금 (10분간)

---

## 2. 팀 관리 영역

### 2-1. 팀 생성 페이지

**목적**
- 새로운 팀을 만들어 협업 공간 구성

**사용자 행동**
- 팀 이름 입력
- 팀 설명 입력 (선택)
- 팀 생성 버튼 클릭

**주요 구성요소**
- 팀명 입력 필드 (필수, 최대 50자)
- 팀 설명 텍스트 영역 (선택, 최대 200자)
- 팀 아이콘 업로드 (선택)
- 생성 버튼

**다음 단계**
- 팀 생성 완료 → 소유자 권한으로 대시보드 이동
- 환영 메시지 및 첫 메뉴얼 작성 가이드 표시

**예외처리**
- 팀명 미입력: "팀 이름을 입력해주세요" 에러 표시
- 중복 팀명: 허용 (팀은 고유 ID로 구분)

---

### 2-2. 팀 관리 대시보드 (소유자 전용)

**목적**
- 팀원 초대, 권한 관리, 팀 정보 수정

**사용자 행동**
- 팀원 초대 (이메일/링크)
- 기존 팀원 권한 변경
- 팀 정보 수정
- 팀원 내보내기

**주요 구성요소**
- **팀원 목록 섹션**
  - 팀원 이름, 이메일, 권한 표시
  - 권한 변경 드롭다운 (소유자/편집자/뷰어)
  - 내보내기 버튼
- **초대 섹션**
  - 이메일 초대 버튼
  - 링크 초대 버튼
  - 초대 대기 목록
- **팀 정보 섹션**
  - 팀명, 설명, 아이콘 수정
- **위험 구역**
  - 팀 삭제 버튼 (빨간색 강조)

**다음 단계**
- 초대 완료 → 초대 메일 발송 또는 링크 복사
- 권한 변경 → 해당 팀원에게 알림
- 팀 삭제 → 모든 데이터 삭제 후 홈으로 이동

**예외처리**
- 편집자/뷰어 접근 시: "권한이 없습니다. 소유자만 접근 가능합니다" 메시지
- 소유자 권한 이전: "소유자 권한을 이전하면 당신은 편집자가 됩니다" 경고
- 팀 삭제: 팀명 입력 확인 모달 (실수 방지)

---

### 2-3. 팀 초대 - 이메일 방식

**목적**
- 특정 이메일 주소로 팀 가입 초대 발송

**사용자 행동**
- 초대할 이메일 주소 입력 (여러 개 가능)
- 부여할 권한 선택 (편집자/뷰어)
- 초대 메일 발송 버튼 클릭

**주요 구성요소**
- 이메일 입력 필드 (쉼표 또는 엔터로 여러 개 입력)
- 권한 선택 라디오 버튼 (편집자/뷰어)
- 초대 메시지 입력 (선택)
- 발송 버튼

**다음 단계**
- 초대 메일 발송 완료 → 팀 관리 대시보드
- 초대 대기 목록에 추가

**예외처리**
- 잘못된 이메일 형식: "유효한 이메일 주소를 입력해주세요"
- 이미 팀원인 경우: "이미 팀에 속한 사용자입니다"
- 초대 중복: "이미 초대를 보낸 사용자입니다 (재발송 가능)"

---

### 2-4. 팀 초대 - 링크 방식

**목적**
- 공유 가능한 초대 링크 생성

**사용자 행동**
- 부여할 권한 선택 (편집자/뷰어)
- 링크 유효기간 설정 (선택)
- 링크 생성 버튼 클릭
- 생성된 링크 복사

**주요 구성요소**
- 권한 선택 라디오 버튼 (편집자/뷰어)
- 유효기간 선택 (1일/7일/30일/무제한)
- 링크 생성 버튼
- 생성된 링크 표시 및 복사 버튼

**다음 단계**
- 링크 복사 후 외부 공유 (메신저, 이메일 등)
- 링크 접속자가 가입 또는 로그인 후 자동 팀 가입

**예외처리**
- 링크 만료: "만료된 초대 링크입니다" 안내 페이지
- 이미 다른 팀 소속: "이미 다른 팀에 속해있어 가입할 수 없습니다" 에러

---

## 3. 대시보드 (메인 화면)

### 3-1. 대시보드 레이아웃

**목적**
- 메뉴얼 관리 및 팀 협업의 중심 화면

**사용자 행동**
- 좌측 사이드바에서 메뉴얼 선택
- 새 메뉴얼 생성
- 팀 관리 메뉴 접근
- 마이페이지 접근

**주요 구성요소**
- **좌측 사이드바**
  - 팀 이름 및 아이콘
  - 메뉴얼 목록 (계층 구조)
  - 새 메뉴얼 추가 버튼
  - 팀 관리 메뉴 (소유자만)
  - 마이페이지 링크
- **중앙 영역**
  - 선택된 메뉴얼 내용 표시
  - 또는 환영 메시지 (메뉴얼 미선택 시)
- **상단 헤더**
  - 검색 바
  - 알림 아이콘
  - 사용자 프로필

**다음 단계**
- 메뉴얼 선택 → 메뉴얼 상세 화면 (읽기 또는 편집)
- 새 메뉴얼 추가 → 메뉴얼 편집 화면
- 팀 관리 → 팀 관리 대시보드
- 마이페이지 → 내 정보 수정

**예외처리**
- 메뉴얼 없음: "첫 메뉴얼을 만들어보세요!" 가이드 표시
- 권한 없는 메뉴얼: 클릭 시 "읽기 권한이 없습니다" 메시지

---

## 4. 메뉴얼 작성/편집 영역

### 4-1. 메뉴얼 편집 화면 (핵심 기능)

**목적**
- 드래그앤드롭 기반으로 직관적인 메뉴얼 작성

**사용자 행동**
- 좌측에서 계층 구조 (대제목/소제목) 추가 및 관리
- 중앙에서 텍스트 및 이미지 블록 추가
- 블록을 드래그로 순서 변경
- 우측에서 공유 및 설정 관리
- 자동 저장 (5초마다)

**주요 구성요소**

**A. 좌측 사이드바 - 계층 구조 관리**
- 메뉴얼 제목 (편집 가능)
- 계층 트리 구조
  - 대제목 (depth 1)
  - 소제목 (depth 2)
  - 하위 소제목 (depth 3, 최대)
- 각 섹션 우클릭 메뉴
  - 하위 섹션 추가
  - 이름 변경
  - 삭제
  - 위/아래 이동
- 드래그로 계층 변경 가능
- 섹션 펼치기/접기 토글

**B. 중앙 편집 영역 - 콘텐츠 블록**
- 블록 추가 버튼 (+)
  - 텍스트 블록
  - 이미지 블록
  - 구분선 블록
  - 코드 블록 (선택)
- 각 블록 좌측에 드래그 핸들 (⋮⋮)
- 블록 호버 시 액션 버튼
  - 복사
  - 삭제
  - 위/아래 이동
- 블록 간 드롭 영역 표시 (파란색 라인)

**C. 우측 설정 패널**
- **메뉴얼 설정 탭**
  - 제목 변경
  - 설명 추가
  - 커버 이미지 설정
- **공유 설정 탭**
  - 팀 내부 공유 (팀원 선택 + 권한)
  - 외부 공유 (링크 생성)
- **버전 이력 탭**
  - 수정 시간 및 작성자 표시
  - 버전 비교
  - 이전 버전으로 되돌리기

**다음 단계**
- 자동 저장으로 실시간 반영
- 나가기 버튼 클릭 → 최종 저장 후 대시보드

**예외처리**
- 편집 권한 없음: 읽기 전용 모드로 전환
- 네트워크 오류: "저장 실패, 재시도 중..." 알림
- 이미지 업로드 실패: "파일 크기가 너무 큽니다 (최대 10MB)" 에러

---

### 4-2. 텍스트 블록 편집

**목적**
- 서식이 적용된 텍스트 콘텐츠 작성

**사용자 행동**
- 텍스트 입력
- 서식 도구로 스타일 적용
- 링크 삽입
- 목록 작성

**주요 구성요소**
- 리치 텍스트 에디터
- 상단 툴바
  - 제목 (H1/H2/H3)
  - 굵게/기울임/밑줄
  - 글머리 기호/번호 목록
  - 링크 삽입
  - 인용구
  - 코드 인라인
- 실시간 미리보기

**다음 단계**
- 블록 외부 클릭 → 편집 완료 및 자동 저장

**예외처리**
- 빈 텍스트 블록: 자동으로 제거하지 않음 (사용자 의도 존중)
- URL 형식 오류: "올바른 링크 주소를 입력해주세요"

---

### 4-3. 이미지 블록 추가

**목적**
- 메뉴얼에 시각 자료 추가

**사용자 행동**
- 이미지 업로드 (파일 선택 또는 드래그앤드롭)
- 이미지 크기 조정
- 정렬 설정 (좌/중앙/우)
- 캡션 추가

**주요 구성요소**
- 이미지 업로드 영역
  - "파일 선택" 버튼
  - "또는 이미지를 드래그하세요" 안내
- 업로드 후 설정 메뉴
  - 크기 조정 슬라이더 (50%~100%)
  - 정렬 버튼 (좌/중앙/우)
  - 캡션 입력 필드
  - 삭제 버튼

**다음 단계**
- 이미지 업로드 완료 → 블록으로 추가
- 계속 작업

**예외처리**
- 파일 형식 오류: "JPG, PNG, GIF 파일만 업로드 가능합니다"
- 파일 크기 초과: "이미지 크기는 10MB 이하여야 합니다"
- 업로드 실패: "업로드에 실패했습니다. 다시 시도해주세요"

---

### 4-4. 블록 드래그앤드롭

**목적**
- 블록 순서를 자유롭게 변경하여 메뉴얼 구조 조정

**사용자 행동**
- 블록 좌측 드래그 핸들 클릭 후 드래그
- 원하는 위치로 이동
- 드롭하여 위치 확정

**주요 구성요소**
- 각 블록 좌측에 드래그 핸들 아이콘 (⋮⋮)
- 드래그 중 블록 반투명 처리
- 드롭 가능 영역 파란색 라인 표시
- 드롭 후 부드러운 애니메이션

**다음 단계**
- 드롭 완료 → 블록 순서 변경 및 자동 저장

**예외처리**
- 잘못된 위치 드롭: 원위치로 되돌림
- 드래그 중 취소 (Esc): 원위치 복귀

---

## 5. 메뉴얼 공유 영역

### 5-1. 팀 내부 공유

**목적**
- 팀원에게 메뉴얼 접근 권한 부여

**사용자 행동**
- 공유할 팀원 선택
- 권한 설정 (편집자/뷰어)
- 공유 버튼 클릭

**주요 구성요소**
- 팀원 목록 (체크박스)
- 권한 선택 라디오 버튼
  - 편집자: 수정 가능
  - 뷰어: 읽기만 가능
- 공유 버튼
- 현재 공유 중인 팀원 목록

**다음 단계**
- 공유 완료 → 해당 팀원에게 알림
- 팀원의 대시보드에 메뉴얼 표시

**예외처리**
- 권한 없는 사용자: "메뉴얼 소유자만 공유 설정을 변경할 수 있습니다"

---

### 5-2. 외부 공유 링크

**목적**
- 팀 외부 사람에게 메뉴얼 공개

**사용자 행동**
- 공유 범위 선택 (제목만/전체 공개)
- 링크 생성 버튼 클릭
- 생성된 링크 복사

**주요 구성요소**
- 공유 범위 선택
  - "제목만 공개": 비회원도 제목 목록만 볼 수 있음
  - "전체 공개": 비회원도 전체 내용 볼 수 있음
- 링크 생성 버튼
- 생성된 링크 표시 및 복사 버튼
- 링크 비활성화 버튼

**다음 단계**
- 링크 복사 후 외부 공유
- 링크 접속자는 로그인 없이 접근 (범위에 따라)

**예외처리**
- 링크 비활성화: "이 링크는 더 이상 유효하지 않습니다" 안내

---

## 6. 버전 관리 영역

### 6-1. 버전 이력 보기

**목적**
- 메뉴얼의 수정 내역 확인

**사용자 행동**
- 버전 이력 탭 클릭
- 시간별로 정렬된 버전 목록 확인
- 특정 버전 선택하여 비교

**주요 구성요소**
- 버전 목록 (최신순)
  - 수정 시간
  - 작성자 이름
  - 변경 요약 (자동 생성)
- 버전 비교 버튼
- 되돌리기 버튼

**다음 단계**
- 버전 비교 → 변경 사항 하이라이트 표시
- 되돌리기 → 이전 버전으로 복원

**예외처리**
- 버전 없음: "저장된 버전이 없습니다"

---

### 6-2. 버전 되돌리기

**목적**
- 잘못된 수정을 이전 버전으로 복원

**사용자 행동**
- 되돌릴 버전 선택
- 되돌리기 버튼 클릭
- 확인 모달에서 확정

**주요 구성요소**
- 버전 선택 체크박스
- 되돌리기 버튼
- 확인 모달
  - "이 버전으로 되돌리시겠습니까?"
  - 현재 내용이 새 버전으로 저장됨 안내
  - 확인/취소 버튼

**다음 단계**
- 되돌리기 완료 → 편집 화면 새로고침
- 현재 버전이 새 버전으로 이력에 추가

**예외처리**
- 편집 권한 없음: "되돌리기는 편집자 이상만 가능합니다"
- 되돌리기 실패: "오류가 발생했습니다. 다시 시도해주세요"

---

## 7. 마이페이지

### 7-1. 내 정보 수정

**목적**
- 사용자 개인 정보 관리

**사용자 행동**
- 이름 변경
- 비밀번호 변경
- 프로필 사진 업로드

**주요 구성요소**
- 프로필 사진 업로드
- 이름 입력 필드
- 이메일 표시 (변경 불가)
- 비밀번호 변경 버튼
- 저장 버튼

**다음 단계**
- 저장 완료 → "정보가 업데이트되었습니다" 알림

**예외처리**
- 이름 미입력: "이름을 입력해주세요"
- 비밀번호 변경 실패: "현재 비밀번호를 확인해주세요"

---

### 7-2. 알림 설정

**목적**
- 알림 수신 환경 설정

**사용자 행동**

**주요 구성요소**
- 알림 항목 토글 스위치
  - 메뉴얼 공유됨
  - 메뉴얼 수정됨
  - 팀원 초대됨
  - 권한 변경됨
- 이메일 알림 수신 여부
- 저장 버튼

**다음 단계**
- 설정 저장 → 알림 정책 업데이트

**예외처리**
- 없음 (설정은 항상 저장 가능)

## 8. 비회원 접근 화면

### 8-1. 공유 메뉴얼 제목 보기 (비회원)

**목적**
- 외부 공유된 메뉴얼의 제목을 통해 흥미 유발

**사용자 행동**
- 공유 링크 접속
- 메뉴얼 제목 목록 확인
- 회원가입 유도

**주요 구성요소**
- 메뉴얼 제목 목록 (계층 구조)
- 내용은 잠금 처리 (흐린 텍스트)
- 상단 안내 배너
  - "전체 내용을 보려면 회원가입이 필요합니다"
  - 회원가입 버튼
  - 로그인 버튼

**다음 단계**
- 회원가입 → 가입 후 전체 내용 접근
- 로그인 → 로그인 후 전체 내용 접근

**예외처리**
- 링크 만료: "이 메뉴얼은 더 이상 공유되지 않습니다"
- 삭제된 메뉴얼: "메뉴얼을 찾을 수 없습니다"

---

### 8-2. 공유 메뉴얼 전체 보기 (비회원)

**목적**
- 외부 공유된 메뉴얼 전체 내용 공개 (전체 공개 설정 시)

**사용자 행동**
- 메뉴얼 읽기
- 스크롤하여 내용 확인

**주요 구성요소**
- 메뉴얼 전체 내용 (읽기 전용)
- 상단 안내 배너
  - "이 메뉴얼은 [팀명]에서 공유했습니다"
  - "직접 메뉴얼을 만들어보세요" 회원가입 버튼

**다음 단계**
- 회원가입 유도

**예외처리**
- 권한 변경: "이 메뉴얼은 더 이상 공개되지 않습니다"

---

**사용자 행동**
- 이전 페이지로 돌아가기

**주요 구성요소**
- 에러 메시지
  - "이 기능은 [필요한 권한]만 사용할 수 있습니다"
- 돌아가기 버튼

**다음 단계**
- 돌아가기 → 이전 화면

**예외처리**
- 없음

---

### 9-2. 네트워크 오류

**목적**
- 일시적 네트워크 문제 안내

**사용자 행동**
- 재시도 버튼 클릭

**주요 구성요소**
- 오류 아이콘
- "인터넷 연결을 확인해주세요" 메시지
- 재시도 버튼

**다음 단계**
- 재시도 → 요청 재전송

**예외처리**
- 3회 재시도 실패: "서버에 문제가 발생했습니다. 잠시 후 다시 시도해주세요"

---

### 9-3. 페이지를 찾을 수 없음 (404)

- 잘못된 URL 접근 안내

**사용자 행동**
- 홈으로 돌아가기

**주요 구성요소**
- "페이지를 찾을 수 없습니다" 메시지
- 홈으로 버튼

**다음 단계**
- 홈으로 → 대시보드 또는 로그인 페이지

**예외처리**
- 없음
