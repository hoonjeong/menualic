# 메뉴얼 제작 서비스 - 기획 산출물 요약

---

## 📋 프로젝트 개요

### 서비스명
**메뉴얼 제작 서비스**

### 핵심 가치 제안
팀 단위로 업무 메뉴얼을 **드래그앤드롭**으로 쉽게 작성하고, **계층 구조**로 체계적으로 관리하며, **안전하게 공유**하는 협업 플랫폼

### 주요 타겟
- 스타트업 및 중소기업 팀장/팀원
- 프로세스 문서화가 필요한 조직
- 지식 공유 문화를 만들고 싶은 팀

### 핵심 차별점
1. **직관적인 편집**: 드래그앤드롭으로 텍스트와 이미지 배치
2. **계층 구조 최적화**: 대제목/소제목/하위 섹션으로 메뉴얼 체계화
3. **권한 기반 협업**: 소유자/편집자/뷰어 권한으로 안전한 협업
4. **버전 관리**: 수정 이력 자동 기록 및 되돌리기
5. **유연한 공유**: 팀 내부 + 외부 공유 모두 지원

## 👥 사용자 페르소나

### 페르소나 1: 김팀장 (소유자)
- **나이**: 35세
- **직책**: 마케팅팀 팀장
- **목표**: 팀원들과 업무 프로세스를 체계적으로 공유
- **불편함**: 
  - 기존 문서 도구는 복잡하고 무거움
  - 팀원 권한 관리 어려움
- **기대 효과**:
  - 간단한 UI로 빠르게 메뉴얼 작성
  - 팀원 권한 세밀하게 관리
  - 외부 공유로 협력사와도 소통

### 페르소나 2: 박 대리 (편집자)
- **나이**: 28세
- **직책**: 마케팅팀 대리
- **목표**: 자신의 업무 지식을 메뉴얼에 추가
- **불편함**:
  - 기존 도구는 편집이 자유롭지 않음
  - 이미지 삽입이 불편함
- **기대 효과**:
  - 드래그앤드롭으로 직관적 편집
  - 이미지와 텍스트 자유롭게 배치

### 페르소나 3: 최 인턴 (뷰어)
- **나이**: 24세
- **직책**: 인턴
- **목표**: 팀의 업무 프로세스 학습
- **불편함**:
  - 메뉴얼을 찾기 어려움
  - 최신 정보인지 확신이 없음
- **기대 효과**:
  - 한곳에 모인 메뉴얼 목록
  - 버전 이력으로 최신 정보 확인

## 🛠 기술 스택 권장사항

### 프론트엔드
- **프레임워크**: React 또는 Next.js
- **드래그앤드롭**: react-beautiful-dnd 또는 dnd-kit
- **리치 텍스트 에디터**: Tiptap 또는 Slate.js
- **상태 관리**: Zustand 또는 Recoil
- **스타일링**: Tailwind CSS

### 백엔드
- **언어/프레임워크**: Node.js (Express) 또는 Python (FastAPI)
- **데이터베이스**: PostgreSQL (메인), Redis (캐시/세션)
- **인증**: JWT + OAuth 2.0
- **파일 스토리지**: AWS S3 또는 Cloudflare R2
- **실시간 통신**: WebSocket (Socket.io)

### 인프라
- **호스팅**: AWS, GCP, 또는 Vercel
- **CI/CD**: GitHub Actions
- **모니터링**: Sentry (에러 트래킹), Google Analytics (사용자 분석)
